ChatGPT提问指南：如何利用AI提高法律服务效率

发表于 2023-03-29 更新于 2023-03-30 分类于 ChatGPT

阅读前的重要提醒

  1. 本指南基于ChatGPT3.5在2023年3月15日前的版本制作。
  2. 律师/法务在使用ChatGPT中所获得的答案都可能存在错误（特别是法律规范检索），请将ChatGPT作为工作、学习的辅助工具，请勿完全依赖ChatGPT。
  3. 请使用文明礼貌的语言向ChatGPT进行提问。

一、关于本指南

（一）什么是ChatGPT

ChatGPT是一种基于人工智能的自然语言处理系统，它可以为用户提供智能问答和自动生成文本的功能。作为一种高度先进的语言模型，ChatGPT已经在各个领域得到了广泛的应用，包括自然语言理解、机器翻译、文本生成等。在不同的应用场景下，ChatGPT都能够为用户提供准确、高效、个性化的服务，并帮助他们快速解决各种问题。

（二）ChatGPT可以为律师/法务工作提供什么帮助

律师和法务工作者可以使用ChatGPT来快速查找法律文献和案例、自动生成法律文件和协议的初稿、快速解决咨询问题以及对客户需求进行聚合分析等，让律师和法务工作者在工作中更加高效和精准，为客户提供更好的服务。

（三）本指南的目的

由于ChatGPT的回答具有开放性和发散性，律师和法务工作者可能会遇到某些问题，如无法准确找到所需的信息或无法理解其回答。因此，本文档的目的是提供针对性的提问指南，帮助律师和法务工作者更好地利用ChatGPT，更快速地获取准确的答案。这些指南旨在提供提问的类型和示例，以便提问者准确地表述问题、避免歧义、调整输入内容以提高ChatGPT回答的准确性等，以充分利用ChatGPT，提供更加高效和精准的法律服务。

二、提问前的准备

（一）ChatGPT的功能与限制

ChatGPT能够自动回答人类的自然语言问题，并生成具有逻辑性和连贯性的语言文本。通过深度学习和更广泛的文本库数据，ChatGPT的表现不断改进，功能也得到不断扩展。但同时，ChatGPT也存在一些限制和潜在的问题，这些需要在提问前被充分认识到。

ChatGPT的功能：

  1. 语言理解：ChatGPT不仅可以回答问题，还能够理解人类语言背后的逻辑结构，理解上下文的信息，并进行情感表达。
  2. 文字生成：ChatGPT可以自动生成语言文本，包括但不限于文章、评论、新闻、博客、小说、诗歌等，并且通常在文本质量上表现出色，多次连续生成文本的成功率也会不断提升。
  3. 知识总结：ChatGPT可以汇总大量的信息、数据和文献，帮助用户总结和了解特定领域的知识，并提供建议。

ChatGPT的限制：

  1. 训练数据：ChatGPT的回答很大程度上取决于其所接受的训练数据质量和覆盖范围，如果训练数据中存在偏差，模型将会出现错误的结果。GPT3.5的数据截止时间为2021年，因此其回答的内容与现在的时间有一定的出入，这种情况在GPT4和New Bing中改善了很多。
  2. 知识遗漏：ChatGPT无法从社交情境中获得的知识，也无法从个人对话中的语言句式中获得知识。
  3. 逻辑结构：ChatGPT能够识别并理解上下文线，但缺乏逻辑推理，导致在某些情况下它所产生的回复显得逻辑不清，甚至毫无逻辑根据。
  4. 质量保障：ChatGPT的产出文本可能存在错误，尤其是涉及敏感和风险控制问题的内容时，需要进行仔细的审核和检查。

（二）制定具体的问题并尽可能明确

制定具体、明确的问题是获取准确答案的关键。提问者需要了解需要的信息、提供详细上下文信息、避免使用含糊不清的词语、注意问题的语言和细节、并尝试自己回答问题，以制定具体明确的问题。
首先，提问者需要明确问题的范围和目标，确定需要哪些类型的信息，需要使用相关的关键字和专业术语以便更好地提问。
其次，提问者需要提供详细的上下文信息。需要清晰地描述所涉及的背景和细节，以使ChatGPT能够更好的理解问题。例如，在法律咨询中，需要描述具体的案件、诉求和相关的法律条款。
第三，提问者需要避免使用模糊、不确定或含糊不清的词语。直接提出需要解决的问题，并明确该问题的所涉及的关键内容。
第四，提问者需要注意问题的语言和细节。问题的语言应恰当、专业，避免语法和拼写错误。在表述问题时，要遵循专业的、文明礼貌的语言规范。
最后，提问者可以尝试自己回答问题。这有助于更清晰地表达问题和更好地理解ChatGPT提供的答案。在提出问题之前，提问者可以思考想要解决的问题，并构思一定的方案，以保证作出问题的陈述更加清楚明白。
在制定问题的过程中，提问者需要遵循上述建议，从而制定具体、明确的问题，进而更好地实现ChatGPT协助。

三、提问的流程

（一）注册和登录ChatGPT账号

略

（二）选择合适的模型和设置参数

首选GPT4和New Bing，不能低于GPT3.5。

（三）输入问题并等待模型生成答案

当使用ChatGPT或其他自然语言处理模型（NLP）解答问题时，一般需要经过以下步骤：

  1. 输入问题：在模型的输入框中输入所要解答的问题。
  2. 文本预处理：模型对输入的问题进行文本预处理，一般会删除停用词、标点符号、特殊字符等，并将语句转换成数字矩阵，以便模型进行计算和分析。
  3. 模型运算：模型会针对输入的问题进行内部运算，使用之前学习的语言知识，寻找匹配度最高的答案，或是从已有语言材料中进行概括得出回答。
  4. 输出答案：模型会返回一个或多个可能性回答。提问者需要对这些答案进行审查，判断哪一个或哪些最符合需求或期望。
    简言之，输入问题后等待答案生成。

（四）评估答案的质量并进一步完善问题

得到初步答案后，提问者需要评估生成的答案的质量，并决定是否需要进一步完善提问的问题。以下是一般的步骤：

  1. 查看答案的准确性：提问者需要检查模型生成的答案是否与问题相关并且是否准确。
  2. 查看答案的清晰度：提问者需要检查答案是否易于理解。如果答案使用专业词汇或难以理解的语言，提问者可以通过提供更多的上下文信息或使用更简单的语言来帮助模型更好地理解问题。
  3. 查看答案的完备性：提问者需要确定模型生成的答案是否完整，并针对回答是否遗漏了某些方面进行修正。
  4. 比较多个答案：如果模型生成了多个答案，提问者需要比较这些答案并决定哪个答案最相关、最准确。在处理复杂问题时，一般需要检查多个答案才能获得完整的信息。
  5. 重新评估原问题：如果模型无法生成准确的答案，提问者需要重新评估原始问题，以确定问题是否有足够信息让模型作出回答。评估问题和提供更详细信息可以提高模型生成的答案的质量。

（五）反复提问直至获得满意的答案

  1. 补充提问：如果提问者对已获得的答案不满意，则可修改或补充问题，并再次尝试输入到模型中。
  2. 再次获得答案：提问者再次对获得的答案进行评估。
  3. 重复前述两个步骤：如果仍然不满意答案，提问者需要重复提问，直到获得自己满意的答案为止。

四、问题/命令示例

（一）法律规范检索

1. 事件或行为可能涉及的法律规定

“某一事件（或者行为）是否涉及某种法律？”
“如果我在公共场合摆摊卖商品，是否需要申请许可证？”
“如果我与某人签订的合同没有明确约定条款，是否仍然有效？”
“如果某人在公共场合辱骂他人，涉及到哪些法律规定？”

2. 法律的具体规定、背景信息等

“某一法律规定的具体条款是什么？”
“依据《民法典》，有关合同有效期的具体规定是什么？”
“某项法律的出台背景和原因是什么？”
“《网络安全法》的出台背景和原因是什么？”
“某项法律的适用范围是什么？”
“《反垄断法》规定的适用范围包括哪些领域或行业？”

3. 对某一法律条文的理解

“请问根据《中华人民共和国合同法》，当事人可以按照约定或者交易习惯确定合同的履行期限。这里所指的交易习惯具体包含哪些方面？”
“请问根据《中华人民共和国劳动合同法》，用人单位应当为劳动者缴纳社会保险费用。这里的社会保险费用具体包括哪些方面？”
“如何理解：食品不符合食品安全标准，消费者主张生产者或者经营者依据食品安全法第一百四十八条第二款规定承担惩罚性赔偿责任，生产者或者经营者以未造成消费者人身损害为由抗辩的，人民法院不予支持”

4. 对法律规范的检索结果列出相应的链接

“请为我检索法律规范相关的链接”或者：”能否给我提供有关法律法规的相关链接？”
“请为我检索中华人民共和国的劳动法相关的链接。”
“能否提供有关商标保护的法律法规链接？”
“我需要了解有关消费者权益保护的法律法规，请为我提供相关链接。”
“可否提供有关婚姻家庭和继承方面的法律法规链接？”
“请为我提供有关企业注销的法律法规链接。”

“我要为公司与另一家企业签订一份商业合作协议，可以提供一份模板或建议，其中需要包括责任、保密、合同期限等方面的条款。”
“我想雇佣某位独立承包商为我工作，需要一份合同模板，以明确雇员与雇主之间的义务和责任，包括报酬、工期、保密等方面的条款。”

“我需要雇佣员工，希望能够提供一份简单易懂、合法有效的雇佣合同模板，其中需要包括工作职责、工资、福利、辞退通知期等方面的内容。”

2. 编写合同条款或其他法律表述

“请编写一条关于解除劳动合同的特殊情况约定，当雇员因突发事件无法履行合同义务时如何处理。场景：雇员在合同期内不幸遭遇事故导致双腿瘫痪，无法再从事原职工作。”
“请编写一条关于产品质量保证的约定，阐述供应商应如何负责确保供应的产品符合合同要求。场景：供应商与采购方签署合同，提供某种机器设备。”
“请编写一条关于个人信息保护的约定，说明如何保护受托方收集到的个人敏感信息。场景：银行服务合同中涉及到客户的身份信息和资产状况等隐私信息。”
“请编写一条关于出租房屋的条款，保证租户按时缴纳租金并维护房屋及设施的约定。场景：业主将其房屋出租给租户，签署房屋租赁合同。”
“请编写一条关于商业机密保护的法律条款，阐述如何保护公司机密信息并禁止员工披露公司机密。场景：某科技公司与员工签署雇佣合同，员工需要签署保密协议。”

3. 生成文档摘要

生成案件事实摘要、编写法律文书中的事实部分
“请按照以下事实，为某起交通事故案件编写摘要和对应法律文书的事实部分：案发时间、地点、当事人姓名、车辆情况、事故经过、事故原因、损失情况等。”
“希望您可以为一起涉及农村土地纠纷的案件生成事实摘要和事实部分，内容包括土地的具体位置、所有人信息、使用情况，争端的缘由和过程等。”
“需要您为一份涉及人身伤害的民事案件生成摘要和事实部分，包括原告、被告信息，事件发生时间、地点，当事人受伤情况，诊治情况，损失赔偿等内容。”
“请为一起涉及违约金的商业合同纠纷案件编写案件事实摘要和相应项目的事实情况，包括合同签订时间、签订双方信息、违约情况、违约金数额计算、相关证据等。”
“请您根据某起婚姻继承案件的事实，为该案件编写事实部分，包括当事人之间的亲属关系、家庭成员信息，被继承人的财产状况及分配情况，继承过程中的争议与解除等内容。”

（三）法律案件处理

1. 案件事实描述

“请用法律术语描述一起劳动纠纷案件中，被告单位因违反劳动合同条款，导致原告无法按时领取工资的具体情况。具体情形为：王三与天天公司签了合同，其中第五条约定了每月五日发工资5000元，但是天天公司每月10日才发工资，并且只发四千元，还剩一千元暂扣至半年才发一次。”

2. 案件争议焦点总结

“请总结一起婚姻纠纷案件中，原告认为被告存在家庭暴力行为，被告认为原告在婚姻中有外遇、不负责任的情况下，争议焦点是什么？”
“对于一起消费者权益保护案件，原告认为被告销售的商品存在质量问题，被告认为原告在购买商品后未按照说明书使用，争议焦点是什么？”
“请总结以下案件的争议焦点：王三与天天公司签了合同，其中第五条约定了每月五日发工资5000元，但是天天公司每月10日才发工资，并且只发四千元，还剩一千元暂扣至半年才发一次，王三不服。”

3. 法律关系分析

“A与B签订了购房合同，但是B迟迟没有办理房产证，A想要解除合同，此时双方的法律关系是什么？”
“一家公司向另一家公司借了一笔贷款，但是没有按期还款，此时两家公司之间的法律关系是什么？”
“请分析以下案件的法律关系：王三与天天公司签了合同，其中第五条约定了每月五日发工资5000元，但是天天公司每月10日才发工资，并且只发四千元，还剩一千元暂扣至半年才发一次，王三不服，遂申请劳动仲裁。”

4. 法律风险分析

“我正在考虑与某公司合作，该公司要求我提供一些敏感信息，但我担心这些信息被滥用或泄露，请问存在哪些法律风险”
“请问天天公司在这个案件中存在怎样的法律风险：王三与天天公司签了合同，其中第五条约定了每月五日发工资5000元，但是天天公司每月10日才发工资，并且只发四千元，还剩一千元暂扣至半年才发一次，王三不服，遂申请劳动仲裁。”

5. 法律措施建议

“我发现我的合作伙伴在未经我的允许下抄袭了我的商标，并在市场上销售类似产品，我该采取什么法律措施呢？”
“我发现我的雇员在公司内部泄露了商业机密，我该采取什么法律措施呢？”
“我和另一家公司签署了一份合同，但对方一直未按照协议履行义务，我该采取什么法律措施呢？”
“请问能否给王三提出法律措施建议：王三与天天公司签了合同，其中第五条约定了每月五日发工资5000元，但是天天公司每月10日才发工资，并且只发四千元，还剩一千元暂扣至半年才发一次，王三不服，遂申请劳动仲裁。”

6. 判决建议

“我现在的工作任务已完成，我想要向我的直接领导提出转岗申请，请为我提供一份转岗申请模板。”

3.3 工作报告

“作为团队领导，每月我需要向高层提交关于项目进展的工作报告，请您提供一些撰写这样的报告的模板。我们是法律团队，每个月的工作分为诉讼、合同、广告审查、咨询处理四个方面。”

“我参加了一场公司的战略规划会议，需要写一份会议纪要。请您提供模板，并给我一些建议，如何组织和汇总会议记录。”

“你能否分享一下你对这篇关于气候变化的文章的看法？它引用了很多数据和研究结果，但是也有一些观点颇受争议，我想听听你的看法。”

“我正在为内部员工的劳动用工法律培训制作材料，你有什么建议？能否提供培训大纲？我需要制作一些能够帮助员工理解特定法律领域的课程材料，这些材料需要简洁明了，面向不具备专业法律知识的读者。”

（五）如何提问

如果不知道某一问题如何向ChatGPT提问，可就问题本身向ChatGPT提问：
“如果我要问ChatGPT：【问题的概述】。我应该怎么提问，请提供示例，示例应尽量具体详细，问题的示例需要描述具体的场景。”

五、注意事项

（一）避免依赖ChatGPT解决敏感或复杂问题

虽然ChatGPT是一种高效的自然语言处理工具，但不可以过度依赖：
首先，如果问题涉及敏感信息，应该避免使用ChatGPT。因为这种工具无法判断问题是否存在敏感信息，而且这些信息可能被保存下来，导致机密信息泄漏。
其次，如果问题越来越复杂或者涉及更多的主观判断，ChatGPT的回答可能并不够满足需求。尽管ChatGPT可以提供很好的答案和建议，但是这些回答基于它已经学习到的语言模型。这就意味着回答内容受限于其未知数据和样本范围，可能没有足够的细节和深入分析。
最后，如果问题涉及重要的决策或动作，提问者应该特别小心。ChatGPT的回答只是一个参考，在决策和策略的考虑中需要注意各种因素以及可能的后果。
简而言之，使用ChatGPT时，提问者应该将其视为辅助工具，而不是唯一的决策依据。

（二）确保问题、答案和记录的保密性

对于法律工作者而言，保护客户的隐私和保密数据的安全至关重要。法律工作者需要意识到使用ChatGPT可能会泄露敏感的客户数据，例如商业秘密、银行账户信息等等。因此，应该避免在ChatGPT中直接提交敏感数据。
